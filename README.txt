UNIVERSO-P-DANI — BASE ESTABLE

Archivos necesarios:
- index.html
- style.css
- script.js

IMPORTANTE:
1. Los tres archivos deben estar en la misma carpeta.
2. No cambies los nombres ni agregues (1), espacios o mayúsculas diferentes.
3. Súbelos a la raíz del repositorio de GitHub.
4. En GitHub: Settings > Pages > Deploy from a branch > main > /(root) > Save.
5. Abre la dirección publicada por GitHub Pages, no el archivo index.html directamente desde el teléfono o la computadora.

Resultado correcto:
- Fondo oscuro.
- Muchas estrellas.
- Tarjeta central con “Para Dani”.
- Botón “Entrar”.
- Mensaje temporal “Universo cargado correctamente”.

Si algo falla, la página mostrará un mensaje rojo en la parte inferior. También puedes abrir la consola del navegador para ver el error exacto.
